## 📖 Descripción
**SeriesFav** es una plataforma diseñada para los amantes del streaming que buscan una experiencia fluida, organizada y, sobre todo, libre de publicidad. Actúa como un centro de gestión centralizado que conecta diversas fuentes de streaming en una interfaz optimizada, eliminando las fricciones y riesgos asociados con la publicidad agresiva.

## 🚀 Características Principales
* **Catálogo Unificado:** Explora contenido de múltiples plataformas en un solo lugar.
* **Gestión de Favoritos:** Organiza tu lista de seguimiento personal para no perderte nunca un estreno.
* **Reproductor Optimizado:** Motor de carga rápida diseñado para una visualización sin esperas.
* **Tecnología Zero-Ads:** Neutralización proactiva de publicidad intrusiva y ventanas emergentes.

## 💻 Instalación (Aplicación de Escritorio - Recomendado)
*Para la experiencia más fluida y sin configuraciones adicionales.*

1. Dirígete a la sección de [Releases](../../releases).
2. Busca la versión más reciente (**Latest**).
3. En la sección **Assets**, descarga el archivo `.zip` correspondiente.
4. Descomprime el archivo en la ubicación de tu preferencia.
5. Ejecuta `SeriesFav.exe`.

## 🛠️ Instalación para Desarrolladores
Si deseas clonar el repositorio y trabajar en el código fuente:

1. Clona el repositorio: `git clone https://github.com/keiriblest/SeriesFav.git`
2. Instala las dependencias: `npm install`
3. Ejecuta la aplicación: `npm start`

## 🌐 Configuración en Navegador (Web)
Si prefieres usar **SeriesFav** desde tu navegador, es necesario combinar nuestra extensión **AdShield** con un bloqueador de contenido estándar para garantizar una navegación limpia.

| Herramienta | Función |
| :--- | :--- |
| **uBlock Origin** | Bloqueo eficiente de banners y elementos globales. |
| **AdShield** | Neutralización de scripts anti-adblock en servidores de video. |

> [!IMPORTANT]
> **Instalación de AdShield:**
> 1. Descarga el repositorio de AdShield desde este mismo lugar.
> 2. Abre `chrome://extensions/` en tu navegador.
> 3. Activa el **Modo de desarrollador** (esquina superior derecha).
> 4. Haz clic en **Cargar descomprimida** y selecciona la carpeta de AdShield.

## 🛡️ ¿Por qué SeriesFav?
Nuestra arquitectura permite que el usuario acceda a múltiples fuentes sin exponerse a los riesgos o la frustración de la publicidad agresiva. Al centralizar la gestión de seguridad a través de nuestra extensión y la aplicación nativa, garantizamos que el enfoque del usuario sea siempre el **contenido**.
